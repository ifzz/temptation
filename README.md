# Temptation 

> 数字货币投资策略管理开源网站

## 功能

- 帮助用户创建／管理／监测自定义的数字货币投资策略（不涉及到任何资金存取操作）
- 搭建平台方便用户能互相观摩、借鉴投资策略，并籍此沟通交流

## 开发计划

### 基本功能

- 脚手架 (done)
- 账户管理 (work on)
- 操作接口设计
- 交易市场对接
- 投资策略管理
- corn 任务调度

### 附加功能

- 收益排行榜
- 货币关键词爬虫
- DNN
